Visual Novel Tentang COVID-19 Untuk Grup WhatsApp Keluarga

Written by octaviyawn with ♥ on 21 April 2020 in 3 min

Seperti kebanyakan cerita dari netizen, grup WhatsApp keluarga menjadi tempat lalu lalang berbagai Hoax, karenanya saya terpikirkan untuk membuat Visual Novel.

Visual Novelnya menceritakan tentang Sari yang menanyakan pertanyaan seputar COVID-19 lalu Pemain (Keluarga di grup WA) memilih jawaban yang kemudian akan dijelaskan oleh Sari mengapa jawaban itu benar/salah, untuk materi seputar COVID-19 saya mengambil dari media di Indonesia maupun WHO.

Visual Novel ini dibuat menggunakan Monogatari. Begini langkah-langkah saya dalam membuatnya,

1. Menyiapkan bahannya dulu

Selain Monogatari, untuk menjalankannya saya menggunakan Web Server for Chrome. Code editor menggunakan Atom. Karakter Sari dibuat oleh Liah, latar belakangnya dibuat oleh mrsiraphol.

2. Bikin Cerita

Sebelum dituangkan ke Atom, saya membuat cerita dahulu menggunakan Google Docs.

3. Mulai Bikin

Karena saya ingin Visual Novelnya langsung main tanpa melalui Menu dahulu maka pada js/options.js.

"ShowMenu": false,

Membuat naskah di js/script.js.
