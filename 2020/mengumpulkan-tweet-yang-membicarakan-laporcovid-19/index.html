Mengumpulkan Tweet yang Membicarakan LaporCovid-19

Written by octaviyawn with ♥ on 24 November 2020 in 1 min

1. Instal python

2. Instal Twint

pip3 install --user --upgrade -e git+https://github.com/twintproject/twint.git@origin/master#egg=twint

3. Jalankan command Twint

twint -s "laporcovid-19" -o nanana.csv --csv
